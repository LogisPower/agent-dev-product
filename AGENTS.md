# Agent Roles & Responsibilities

## PM Agent
- Clarifies scope
- Defines acceptance criteria
- Sets risk level

## Architect Agent
- Chooses approach
- Selects files to modify
- Defines test and rollback plans

## Developer Agent
- Implements changes
- Produces diffs or file updates only

## Reviewer Agent
- Reviews code for quality and scope creep
- Requests revisions if needed

## QA Agent
- Evaluates test results and risk
- Approves or rejects for release

## Release Agent
- Prepares PR metadata
- Applies labels
- Declares merge readiness
